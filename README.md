# Git Stats - Git 多仓库可视化分析工具

一个基于 Electron + Vue 3 + TypeScript 开发的桌面应用程序，用于分析和可视化多个 Git 仓库的统计数据。

## ✨ 功能特性

### 📁 多仓库扫描

- 扫描指定目录下的所有 Git 仓库
- 自动识别有效的 Git 仓库
- 显示仓库基本信息（分支、最新提交、作者列表等）
- 支持批量选择和操作

### 📊 仓库分析

- **提交统计**：查看仓库的提交历史、作者贡献度
- **每日提交趋势**：使用 ECharts 可视化展示每日提交数量
- **代码变更统计**：统计新增、删除、净增代码行数
- **作者排名**：按提交数量排序显示各作者的贡献占比

### 👤 个人贡献统计

- **跨仓库统计**：统计指定账号在多个仓库中的总贡献
- **总体概览**：总提交数、涉及仓库数、活跃天数
- **每日提交趋势图**：可视化展示提交活动的时间分布
- **年份数据对比**：对比不同年份的贡献数据，包括：
  - 提交次数对比
  - 新增/删除/净增代码行数对比
  - 新增代码最多/最少月份
  - 提交过的仓库数
  - 新增代码最多仓库
  - 所有对比项都显示变化量和百分比

## 🚀 快速开始

### 环境要求

- Node.js >= 18
- pnpm >= 8（推荐）或 npm/yarn
- Git（用于分析 Git 仓库）

### 安装依赖

```bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install
```

### 开发模式

```bash
# 启动开发服务器
pnpm dev

# 或
npm run dev
```

### 构建应用

```bash
# 构建所有平台
pnpm build

# 构建 Windows 版本
pnpm build:win

# 构建 macOS 版本
pnpm build:mac

# 构建 Linux 版本
pnpm build:linux
```

## 📖 使用指南

### 1. 扫描仓库

1. 启动应用后，点击"选择目录"按钮
2. 选择包含 Git 仓库的目录
3. 应用会自动扫描该目录下的所有 Git 仓库
4. 扫描完成后，会显示找到的仓库列表

### 2. 查看仓库详情

1. 在仓库列表中点击任意仓库
2. 查看该仓库的详细统计信息：
   - 提交历史
   - 作者贡献统计
   - 每日提交趋势图
   - 代码变更统计

### 3. 分析个人贡献

1. 在仓库列表中选择一个或多个仓库（可全选）
2. 输入要分析的 Git 账号名称
3. 点击"统计贡献"按钮
4. 查看该账号在所有选中仓库中的贡献统计：
   - 总体统计概览
   - 每日提交趋势图
   - 年份数据对比（可选择不同年份进行对比）
